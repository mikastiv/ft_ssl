# ft_ssl

a reimplemetation project of part of the OpenSSL program

## Features

#### Standard

- GenRsa
- Rsa
- RsaUtl

#### Hashing

- MD5
- SHA256
- SHA224
- SHA512
- SHA384
- WHIRLPOOL

#### Cipher

- Base64
- DES
- Triple DES

##### With different block cipher modes:
- Electronic Codebook (ECB)
- Cipher block chaining (CBC)
- Propagating cipher block chaining (PCBC)
- Cipher feedback (CFB)
- Output feedback (OFB)
